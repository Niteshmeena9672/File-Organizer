# File-Organizer
A simply implemented File Organizer using NodeJs.


Please follow given steps:-

1. Install NodeJs 
2. Open main.js file in vs code run it
3. Please type (npm init -y) to create package.json file
4. please type int end of package.json file
"bin":{
    "file":"main.js"
  }

5. open terminal type "npm link"

if package.json file is already available just follow step 5 after installing nodeJs
